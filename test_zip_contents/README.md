# 高考金牌私教 - AstrBot 插件 (astrbot_plugin_gaokao)

这是一个能够陪伴你度过高三备考岁月的 AI 学习辅助插件。它基于强大的 GAOKAO-Bench 题库，为你提供全科（语数英物化生政史地）的随机刷题、主观题自动批改、AI 深度错题解析与维度评分追踪。

## 🌟 核心特色
- **海量全真模拟题库**：包含近12年共近3000道高考各省真题（含主观题与客观题）。
- **AI 智能批改与点评**：抛弃刻板的完全匹配比对！AI 名师不仅为你批改主观题，还会给你指出拉分要点。
- **全方位大纲梳理**：遇到不懂的知识点？直接让AI为你生成从定义到经典题型的系统化总结。
- **LaTeX 完美支持**：借助基于 MathJax 的 HTML 渲染引擎，再也不用担心复杂的数学物理公式在聊天软件中变成乱码，我们会以高清图片形式交付题目。
- **数据追踪与薄弱点诊断**：系统会根据年份、题型和科目记录你的得分情况。

## ⚙️ 个性化配置
在 AstrBot 的 WebUI 或 `.env` 中可修改部分配置：
- `gaokao_date`: 设定你实际的高考日期（如 `2026-06-07`），用于精确倒计时。
- `render_as_image`: 若你的客户端不支持原生 LaTeX，保持该开关打开即可将包含复杂公式的题目自动转换为图片发送。
